#### Step 2 - 开发融合器核心逻辑

**原理**：互补滤波器（Complementary Filter），用视觉低频绝对观测校正 IMU 高频相对积分。

**公式**：
```
Q_fused = Slerp(Q_imu, Q_vis, alpha)

alpha = f(C):
  C >= 0.7  ->  alpha = 0.15   # 视觉可靠，IMU 主导高频
  0.5 <= C < 0.7  ->  alpha = 0.08  # 视觉模糊，信任 IMU
  C < 0.5  ->  alpha = 0.0     # 遮挡/出画，纯 IMU

恢复过渡: alpha += 0.01 per frame until alpha = 0.15
```

- [ ] 视觉丢帧时降级到纯 IMU

#### Step 3 - 重置/重新计算机制

**原理**：当视觉观测到来时，将 IMU 累积误差归零。视觉观测提供绝对基准。

## 技术原理附录

### 四元数积分（IMU 基础，当前由 STM32 固件实现）

陀螺仪角速度 omega -> 四元数变化率：
```
dq/dt = 0.5 * q * omega
```
其中 omega = [0, omega_x, omega_y, omega_z]，q 为当前姿态四元数。

离散形式（一阶龙格-库塔）：
```
q_{t+1} = q_t + dq/dt * dt
q_{t+1} = q_t + 0.5 * q_t * omega * dt
```

### Mahony 滤波器（由 STM32 运行）

融合陀螺仪与加速度计，互补滤波在频域：

加速度计提供低频姿态参考（通过重力向量推算俯仰/横滚）：
```
theta_acc = atan2(a_y, a_z)        # 横滚
phi_acc = atan2(-a_x, sqrt(a_y^2 + a_z^2))  # 俯仰
```

Mahony 通过 PI 反馈补偿陀螺仪 bias：
```
omega_bias += K_p * e + K_i * integral(e) * dt
q_{t+1} = q_t + 0.5 * q_t * (omega_measure - omega_bias) * dt
```
其中 e 为加速度计观测与预估重力向量的叉积误差。

### 互补滤波器（本计划核心算法）

| 数据源 | 频率特性 | 累计漂移 |
|--------|---------|---------|
| IMU 四元数 | 高频好（100Hz） | 长期漂移（deg/min） |
| 视觉（HAMER） | 低频（15-30Hz） | 无长期漂移 |

融合公式：
```
Q_fused = (1-alpha) * Q_IMU + alpha * Q_vis   （通过 Slerp 插值）
```
本质等效于频域互补：
```
theta_fused(s) = s/(s+omega_c) * theta_IMU(s) + omega_c/(s+omega_c) * theta_vis(s)
```
其中 omega_c 为截止角频率，alpha 与采样频率和 omega_c 相关。
